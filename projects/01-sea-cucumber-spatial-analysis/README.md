# Spatial Analysis of Sea Cucumber Distribution Using QGIS

## Project Overview

This undergraduate research project examines the distribution of *Holothuria atra* in a coastal ecosystem using ecological observations, spatial analysis, and GIS-based mapping.

The project was developed as part of my Biology undergraduate thesis at the University of Bangka Belitung.

## Research Objective

The main objective is to identify spatial distribution patterns and explore how habitat and tidal conditions may influence the occurrence of *Holothuria atra*.

## Tools

- QGIS
- R
- Microsoft Excel
- Geographic Information Systems
- Ecological data analysis

## Workflow

1. Organize and validate ecological observation data.
2. Review coordinate format and spatial reference system.
3. Import observation points into QGIS.
4. Add relevant coastal and habitat layers.
5. Visualize species distribution.
6. Compare occurrence patterns with environmental or tidal context.
7. Analyze supporting data in R and Excel.
8. Interpret results and document limitations.
